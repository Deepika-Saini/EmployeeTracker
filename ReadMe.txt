Problem Statement:
Creating a UI that helps the management in keeping a track of their employees and their ongoing projects. It also should tell them who is sitting where.
Hovering over a seat should display details of employees in a tooltip.
Also, search bar at the top would help in searching employees through their names and would highlight the search results with blue colored seats.

Steps to run the app:
1. Open terminal/command prompt on your system
2. Use CD command to navigate to your project directory
3. Run command:
npm install
This will install the required dependencies
4. To finally run the project, run command:
npm start 
This will run the project on your browser

Assumptions:
1. Each table has 4 chairs, 2 at top and 2 at bottom
2. There can be n number of table/chairs combinations
3. Consumed data from sample json file
